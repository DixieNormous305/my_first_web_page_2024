Computer Science Degree Overview

This page provides a detailed overview of the Computer Science degree, including its purpose, structure, and a comprehensive list of subjects covered in the program.

Degree Structure

The Computer Science program is designed to equip students with foundational knowledge and practical skills in computing. Below is an overview of the subjects categorized by year:

Year Subject Category
1 Introduction to Programming Core
1 Discrete Mathematics Core
2 Data Structures Core
2 Computer Architecture Core
3 Operating Systems Core
3 Algorithms Core
4 Artificial Intelligence Elective
4 Cybersecurity Elective

Subject Descriptions

  • Introduction to Programming: Learn the basics of programming languages, problem-solving, and algorithmic thinking.
  • Discrete Mathematics: Explore the mathematical foundations of computer science, including logic, set theory, and graph theory.
  • Data Structures: Understand how to organize and manage data efficiently using various structures like arrays, linked lists, and trees.
  • Computer Architecture: Gain insights into the design and functioning of computer systems, from processors to memory management.
  • Operating Systems: Study the principles of operating systems, including process management, scheduling, and file systems.
  • Algorithms: Learn advanced techniques for designing and analyzing algorithms to solve computational problems.
  • Artificial Intelligence: Explore the concepts and techniques used to create intelligent systems, including machine learning and neural networks.
  • Cybersecurity: Understand the principles of securing computer systems and networks against threats and vulnerabilities.
